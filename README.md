# Global Workflow Infrastructure for AWS  

This repo will assist in building an cloud native HPC environment using AWS Parallelcluster 3.13.2.

# Prerequisites: 
Hashicorp Packer 1.15 (https://developer.hashicorp.com/packer/docs)

AWS Parallelcluster 3.13.2 (https://docs.aws.amazon.com/parallelcluster/latest/ug/install-v3.html) 

Amazon Web Services account
#

Using Packer the end user can create a custom AMI with Spack-Stack 1.9.2 packages/libraries needed to build the various application components of Global Workflow
